# Happy Addition Garden

Static math game prepared for GitHub Pages.

## Files

- `index.html`
- `style.css`
- `game.js`
- `.nojekyll`

## Publish On GitHub Pages

1. Create a GitHub repository and push these files to the default branch.
2. Open the repository on GitHub.
3. Go to `Settings` -> `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select your default branch and the `/ (root)` folder.
6. Save.
7. GitHub will publish the site at:

   `https://<your-github-username>.github.io/<your-repo-name>/`

## Local Preview

Open `index.html` in a browser.

## Notes

- The site uses relative paths, so it works on GitHub Pages without build tooling.
- `.nojekyll` ensures GitHub Pages serves the files as a plain static site.
